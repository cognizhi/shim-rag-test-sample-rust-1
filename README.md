# shim-rag-test-sample-rust-1

Synthetic Rust utility crate used to validate SydeKx plan 091 L2-RS
(Rust occurrence extraction + use-declaration resolution). Exercises:
plain / grouped / aliased / wildcard `use` declarations, crate-relative
paths, associated-function calls (`Type::assoc`), method calls with a
self receiver, and strictly positional call shapes.
